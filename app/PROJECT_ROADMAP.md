# Project Roadmap

## Phase 1 - Foundation
- Preserve existing authentication and routing.
- Keep Firestore collections modular and production-ready.
- Strengthen shared layout, theme handling, and page shell.

## Phase 2 - Product Experience
- Expand dashboard, scheduler, settings, and AI studio surfaces.
- Add reusable UI components and stateful empty/loading patterns.
- Prepare toast-based feedback and future admin architecture.

## Phase 3 - Future Integrations
- Add TikTok OAuth and publishing API when approved.
- Introduce OpenAI-backed caption generation behind a server-safe abstraction.
- Extend to multi-workspace and billing flows.
